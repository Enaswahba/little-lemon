# Little Lemon Web App

A React-based single-page application for the Little Lemon restaurant. Users can view the homepage, browse specials, read testimonials, and book a table online with form validation and accessibility support.

## Features

- Responsive UI built with React components
- Booking form with HTML5 and React validation
- API integration to fetch available reservation times
- Booking confirmation page
- Accessibility improvements (ARIA labels, semantic HTML)
- Unit tests for booking form functionality
